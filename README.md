# Cricket Data Analytics

A data analytics project that transforms raw cricket data into interactive, real-time dashboards using Python and Power BI, providing actionable insights into player and team performance.

## Overview

Cricket offers a wealth of statistical data that can reveal performance trends and tactical insights. This project leverages **Pandas** and **NumPy** for data cleaning, preprocessing, and transformation, and **Power BI** for advanced data modeling and interactive dashboard creation.

The project explores multiple analytical dimensions — player performance, match statistics, and trend analysis — to help cricket enthusiasts, analysts, and strategists make informed, data-driven decisions through structured visualization and dynamic filtering.

## Dataset

The dataset used for this project was obtained from **Code Basics**, a structured source for cricket analytics data.

- **Source**: [Code Basics](https://codebasics.io/)
- **Format**: JSON files containing structured records of matches, players, scores, and statistics
- **Usage**: Parsed and processed using Python to build a clean, analysis-ready dataset for visualization

## Outcomes

- Delivered a comprehensive, role-based player evaluation system covering anchors, finishers, all-rounders, and specialist fast bowlers
- Enabled real-time data updates and remote accessibility through Power BI Service, removing the need for manual refreshes
- Built advanced visualizations (scatter plots, bar/line charts) for tracking batting and bowling performance trends
- Supported strategic decision-making by surfacing data-backed insights for batting order, bowling tactics, and match planning
- Improved user engagement through interactive filters enabling dynamic, real-time comparisons
